# 🤖 Basic Chatbot - Python

## 📌 Project Overview

This project is a simple **rule-based chatbot** developed using Python as part of the **CodeAlpha Python Programming Internship**.

The chatbot interacts with users through the console and responds to predefined inputs such as greetings and basic questions. It continuously accepts user input and provides responses until the user types **"bye"** to end the conversation.

This project helps in understanding fundamental Python concepts such as loops, conditions, functions, and user input handling.

---

## 🎯 Objective

The objective of this project is to create a simple chatbot that:

- Accepts text input from users
- Identifies predefined words or sentences
- Responds with predefined answers
- Continues interaction until the user exits

---

## ✨ Features

✔ Interactive console-based chatbot  
✔ Responds to greetings like "hello"  
✔ Answers simple questions  
✔ Uses predefined rules and responses  
✔ Continuous conversation using loops  
✔ Exit option using "bye" command  
✔ Handles unknown inputs gracefully  
✔ Beginner-friendly implementation
